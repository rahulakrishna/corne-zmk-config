# Corne XIAO v2 — Keymap Reference

Miryoku layout on Colemak. 8 layers, all activated via thumb hold. Outer column (col 0 left / col 5 right) is defined only on the base layer and falls through on all others.

---

## Thumb Cluster

```
┌─────────┬─────────┬─────────┐     ┌─────────┬─────────┬─────────┐
│ ESC/MED │ SPC/NAV │ TAB/MOU │     │ ENT/SYM │ BSP/NUM │ DEL/FUN │
└─────────┴─────────┴─────────┘     └─────────┴─────────┴─────────┘
  tap/hold   tap/hold   tap/hold       tap/hold   tap/hold   tap/hold
```

| Thumb | Tap | Hold |
|-------|-----|------|
| Left outer | Esc | Media |
| Left middle | Space | Nav |
| Left inner | Tab | Mouse |
| Right inner | Enter | Sym |
| Right middle | Backspace | Num |
| Right outer | Delete | Fun |

## Home Row Mods (GACS order)

Hold the key for the modifier, tap for the letter.

```
┌─────┬──────────┐     ┌─────┬──────────┐
│ KEY │ HOLD     │     │ KEY │ HOLD     │
├─────┼──────────┤     ├─────┼──────────┤
│  A  │ ⌘  GUI  │     │  N  │ ⇧  SHIFT │
│  R  │ ⌥  OPT  │     │  E  │ ⌃  CTRL  │
│  S  │ ⌃  CTRL │     │  I  │ ⌥  OPT   │
│  T  │ ⇧  SHIFT│     │  O  │ ⌘  GUI   │
└─────┴──────────┘     └─────┴──────────┘
  LEFT HAND               RIGHT HAND
```

Settings: `tap-preferred` flavor · 200 ms tapping term (Miryoku defaults).

## Layer 1 — Nav

Hold **SPC**.

```
┌────────┬──────┬──────┬──────┬──────┬──────┐                    ┌──────┬──────┬──────┬───────┬──────┬──────┐
│        │      │      │      │      │      │                    │ Redo │Paste │ Copy │  Cut  │ Undo │      │
├────────┼──────┼──────┼──────┼──────┼──────┤                    ├──────┼──────┼──────┼───────┼──────┼──────┤
│        │  ⌘   │  ⌥   │  ⌃   │  ⇧   │      │                    │  ←   │  ↓   │  ↑   │   →   │ CAPS │      │
├────────┼──────┼──────┼──────┼──────┼──────┼────────┬───────────┼──────┼──────┼──────┼───────┼──────┼──────┤
│        │      │      │      │      │      │        │           │ HOME │ PGDN │ PGUP │  END  │ INS  │      │
└────────┴──────┴──────┼──────┼──────┼──────┼────────┴───────────┼──────┼──────┼──────┼───────┴──────┴──────┘
                       │      │ (hd) │      │                    │ ENT  │ BSP  │ DEL  │
                       └──────┴──────┴──────┘                    └──────┴──────┴──────┘
```

- Clipboard uses ⌘ (macOS): Undo=⌘Z · Cut=⌘X · Copy=⌘C · Paste=⌘V · Redo=⇧⌘Z
- Left hand home row provides one-handed modifier access for nav combos (e.g. ⇧→ to select)

## Legend

| Symbol | Meaning |
|--------|---------|
| `(hd)` | This key is being held to reach this layer |
| `KEY/⌘` | Tap = letter/key · Hold = modifier or layer |
| `☰` | Hold to activate Button layer |
| `LCK/RCK/MCK` | Left / Right / Middle click |
| `MS_←↓↑→` | Mouse cursor movement |
| `WH_←↓↑→` | Scroll wheel |
